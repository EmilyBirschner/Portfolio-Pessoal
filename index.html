Projetos Realizados

Aqui estão alguns dos projetos que já tirei do papel. É só clicar pra dar uma olhada!

Projeto de App de Clima

Aplicativo web desenvolvido com HTML, CSS e JavaScript para exibir informações climáticas em tempo real. Através da integração com a API gratuita da OpenWeatherMap, o usuário pode pesquisar qualquer cidade e visualizar dados como temperatura, condição do tempo, umidade e velocidade do vento de forma simples e rápida.

Projeto de Checklist

Aplicativo web criado com HTML, CSS e JavaScript para gerenciamento de tarefas. Permite adicionar novas atividades, marcar como concluídas ou pendentes e excluir tarefas da lista. O projeto utiliza Local Storage para salvar os dados no navegador, garantindo que a lista permaneça disponível mesmo após fechar ou atualizar a página.

Projeto de Gerador de Currículos

Aplicativo web desenvolvido com HTML, CSS e JavaScript para gerar currículos personalizados. O usuário preenche seus dados, adiciona experiências e habilidades, e o sistema gera um PDF pronto para download. As informações são salvas automaticamente no navegador, facilitando edições futuras.

Projeto de Gerador de QR Code

Aplicativo web interativo construído com HTML, CSS e JavaScript para a criação e download de QR Codes. O utilizador pode inserir qualquer link ou texto, e a ferramenta gera instantaneamente a imagem do QR Code correspondente. O projeto se aprofunda na manipulação do DOM e na integração de bibliotecas externas (qrcode.js e jsPDF) para oferecer funcionalidades de download em formato JPG e PDF. Além disso, foi dado um foco especial na experiência do utilizador, com um design de tema escuro, mensagens de erro dinâmicas e animações suaves para uma interação mais fluida.
